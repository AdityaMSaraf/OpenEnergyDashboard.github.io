Plans for the Open Energy Dashboard

While OED has been public since March 2017, we have not yet had a major public release. To coincide with AASHE 2017, we will be doing our first release in mid-October. Please see the features page to see what that release can do.

The OED project already has features we hope to release in 3-6 months (January-March 2018). These include:

  • Generalization of energy types beyond electricity including steam, gas, etc.
  • An admin panel to allow someone responsible for the OED system to import meters, enter information needed, and generally control the way OED functions at their site.
  • Allow energy to be represented in other units such as CO2, laundry loads, etc.
  • Have a virtual energy usage page to allow someone to understand the cost associated with using different types of appliances given the time per use, how often and how long used, and the number of people who use those appliances.
  • Make the OED website controls and basic wording multilingual.
  • Provide email warnings of system issues.

We expect to include other features in the next release. You can influence our plans by becoming involved.

We also have a list of feature requests that we have yet to decide when to implement.
