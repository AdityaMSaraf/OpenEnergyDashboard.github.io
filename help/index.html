Help Overview

These pages provide help about the Open Energy Dashboard (OED) system. You can use the links at the top or any help page to go the appropriate section for your question or look at the description/links below.

  • Graph Type: describes the types of graphics and how to select them. (needs images)
  • Graphing Meters: describes the ideas of meters and how to get them graphed. (needs images)
  • Graphing Groups: describes the ideas of groups and how to get them graphed. (needs images)
  • Line Graphic: describes how the line graphic works. (needs images)
  • Bar Graphic: describes how the bar graphic works. (needs images)
  • Compare Graphic: describes how the comparison graphic works. (needs images)
  • Map Graphic: describes how the map graphic works. (todo)
  • Exporting Data: describes how to export data for use outside of OED. (todo)
  • Chart Link: describes how to get a persistent URL for recreating a graphic with a hyperlink. (todo)
  • Hide Options: describes how dock/undock the buttons/options on the left side a hyperlink. (needs images)
  • Page Choices: describes different pages available and choosing them. (todo)
  • Meter Viewing: describes the page where you can see details on meters. (todo)
  • Group Viewing: describes the page where you can see details on groups. (todo)
  • Home Navigation: describes how to return to the main (graphic) OED page. (needs images)
  • Admin User: describes the role, capabilities and usage of admin features. Note these are only available to select people who oversee the OED site so these features are not usually of interest to a general user. (todo)
