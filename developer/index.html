OED Developer Overview

These pages provide information that is useful for OED code developers. It does not contain information that is useful for using or running OED. Please see the main OED website and help pages for that (linked above).

There are developer pages for:

  • Getting Started: how to begin coding development with OED.
  • Our Contributor's License agreement (CLA): describes the advantages, what you will agree to and how to do this step.
  • OED test data: describes how to load sample meter data into your OED system.
  • Code documentation guidelines: describes our desires in documenting code.
  • Pull Requests: describes how to request your work be integrated into the OED project.
  • OED technology learning resources: describes resources to learn about the technologies used in OED.
  • Code Organization: describes the directory structure and meaning of files in the OED code base.
  • Technology Overview: overview of technology usage that currently describes how front-end code becomes our web-based app.
  • OED Developer details: more detailed information that developers may find useful.
  • OED code testing: information on how to use and include test code in OED.

If you are a developer interested in an internship we have a page with further information.
