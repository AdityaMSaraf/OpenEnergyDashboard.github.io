Full Install Output

When you install OED without already have a database or node modules, it does all the steps. Below is typical output for this process. Please note:

  • This output was created on 5 Feb 2021 on a Mac. The exact output will vary with machine type and changes to OED.
  • There are often colors in the output but that is not shown.
  • There may be a delay of many minutes before the line
    web_1 | > fsevents@1.2.12 install /usr/src/app/node_modules/fsevents
    due to the install process downloading all the needed packages. The time needed varies by internet connection, your machine speed and the speed that the repos respond.
  • The warning:
    WARNING: could not open statistics file "pg_stat_tmp/global.stat": Operation not permitted
    may or may not show up. It also shows up at other times. The complete answer is not known but it appears to be a transient error in PostgreSQL writing log information. It has never caused any problems.
