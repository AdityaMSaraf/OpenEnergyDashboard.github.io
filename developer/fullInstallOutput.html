Full Install Output

When you install OED without already have a database or node modules, it does all the steps. Below is typical output for this process. Please note:

  • This output was created on 24 April 2022 on a Mac. The exact output will vary with machine type and changes to OED. This output was what was shown in the nohup.out file for a VSC install.
  • The output and information in install output without redoing database or npm install is similar to what will happen here. The major difference is in what is shown in nohup.out below. Note that if you have run OED previously then the previous output will be before the new output.
  • For unknown reasons, the OED install fails to run at times. See the no npm install output to see the known solution.
