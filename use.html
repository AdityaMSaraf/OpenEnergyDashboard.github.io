Using Open Energy Dashboard

The OED project will be forever free and anyone is welcome to use it. Since the system is under active development, we strongly encourage you to contact us to use the system. The possible ways to use the OED system are:

  • Install the system at your own site. At this time it is best to contact us for help in doing this. This is a good option if you want to keep your data isolated to your own systems.
  • For testing and special reasons, you might be able to temporarily host a site on our server. Please contact us about this.
  • Download and install the software yourself. We are an open source project so you can get the software at any time. You can also modify the software for your own uses but must live by the terms of our license that generally requires you to share what you develop for the good of everyone. Except for those working on the team of developers, we encourage you to consider one of the two options above at this time.

Note you can install OED via Docker which helps automate this process. We are happy to help with initial setup of this process at your site.
