Using Open Energy Dashboard

The OED project will be forever free and anyone is welcome to use it. We encourage you to contact us if you are using OED so we can let you know about significant developments and to help you get the most out of OED. The possible ways to use the OED system are:

  • Install the system at your own site (either your server or in the cloud). The project has installation directions and help on administering OED. Feel free to contact us for help in doing this.
  • For testing and special reasons, you might be able to temporarily host a site on our server. Please contact us about this.
