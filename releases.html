Release Notes

For more information on the features discussed on this page see our feature page.

Version 0.2

  • Added ability to graph a meter as a line or bar graph.
  • Added ability to compare a meter's energy usage over a one week interval.
  • Added ability to create, edit, and delete groups of meters.
  • Can now graph a group as a line or bar graph.
  • Can now compare a group's energy usage over a one week interval.
  • Can now graph and compare multiple meters and groups simultaneously.
  • Added ability to link to specific charts.
  • Meters and groups now have distinct colors when graphed.
  • Improved mobile support.
  • Added ability to export readings from system.
  • Backend improvements for increased speed, stability, and scalability.
  • Improved meter import process.
  • Other minor improvements and fixes.

Upcoming

The system is under active development and we hope to add the following features in the next few months (January-March 2018):

  • Generalization of energy types beyond electricity including steam, gas, etc.
  • An admin panel to allow someone responsible for the OED system to import meters, enter information needed, and generally control the way OED functions at their site.
  • Allow energy to be represented in other units such as CO2, laundry loads, etc.
  • Make the OED website controls and basic wording multilingual.
  • Provide email warnings of system issues.
  • Add the ability to baseline meters and groups.

You can influence our upcoming features by becoming involved. We also have a list of feature requests that we have yet to decide to implement that can be seen here.
